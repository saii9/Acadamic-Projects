Problem-1 (10 Points)

The following parameters have been used to generate a PWM signal. Determine the frequency and duty cycle (in %) of the PWM signal.

 

Fosc = 10MHz, PR2 = 110, TMR2 Prescale Value = 16 and CCPR1L:CCP1CON<5:4> = 80.

 

Answer

Tosc = 1/10 micro  sec = 0.1 micro sec.

PWM Period             = (PR2 + 1)*4*Tosc* (TMR2 Prescale value)

= 111*4*0.1*16 micro sec. = 710.4 micro sec.

PWM Frequency = (1/0.7104) KHz = 1.408 KHz

              

PWM Duty Cycle = CCPR1L:CCP1CON<5:4>*Tosc*(TMR2 Prescale value)

                                    = 80*0.1*16 = 128 micro secs

                                    = 100*128/710.4 = 18.02%

Problem-5: Flexray Problem (15 Points)

The total length of a FlexRay communication cycle is 60 msec, and there are 10,000 macroticks in each communication cycle. There are four nodes: nodes 1, 2, 3 and 4 in the system. The oscillator frequencies of nodes 1, 2, 3 and 4 are 500MHz, 250MHz, 1GHz and 500 MHz, respectively. The number of local clock cycles (oscillator cycles) per microtick of nodes 1, 2, 3 and 4 are 2, 1, 2 and 1, respectively.

a)      Determine the number of microticks per macrotick for each node.

b)      Due to the clock drifts among various nodes (including the Sync Nodes), nodes 3 and 4 detected, at a certain time, that the length of the communication cycle changed by +4% and 4%, respectively. After the clock correction, determine the number of microticks per macrotick for nodes 3 and 4.

 

Answer

The length of each macrotick = 60 msec / 10,000 = 6 micro sec. = 6000 nanosec.

a)      Node 1: One microtick = 2*(1/500) microsec = 4 nanosec

Node 2: One microtick = (1/250) microsec = 4 nanosec

Node 3: One microtick = 2*(1/1000) microsec = 2 nanosec

Node 4: One microtick = (1/500) microsec = 2 nanosec

- - -

Node 1: One macrotick = 6000/4 = 1500 microticks

Node 2: One macrotick = 6000/4 = 1500 microticks

Node 3: One macrotick = 6000/2 = 3000 microticks

Node 4: One macrotick = 6000/2 = 3000 microticks

 

b)      Node 3: The clock of this node is running 4% faster than the average value of the clocks of the Sync Nodes. Thus, for Node 3, the number of  microticks per microtick for should be increased by 4%. Hence, after clock correction, there are 3000*1.04 = 3120 microticks/macrotick for Node 3.

 

Node 4: The clock of this node is running 4% slower than the average value of the clocks of the Sync Nodes. Thus, for Node 4, the number of  microticks per microtick for should be decreased by 4%. Hence, after clock correction, there are 3000*0.96 = 2880 microticks/macrotick for Node 4.

Problem-7 (10 Points)

We would like to generate a 4KHz PWM signal with 25% duty cycle. Determine the contents of PR2 and CCPR1L:CCP1CON<5:4> assuming that Fosc = 10MHz and TMR2 Prescale Value = 16.

 

Answer

Tosc = 1/10 micro  sec = 0.1 micro sec.

PWM Period = 1/4 msec = 250 micro sec.

PWM Period             = (PR2 + 1)*4*Tosc* (TMR2 Prescale value) = 250 micro sec.

i.e. (PR2 + 1)*4*0.1* 16 = 250. Hence PR2 = 250/6.4  1 = 38.0625 = 38

 

PWM Duty Cycle (25%) = 250*0.25 = 62.5 micro sec.

Hence, CCPR1L:CCP1CON<5:4>*Tosc*(TMR2 Prescale value) = 62.5 micro sec

i.e. CCPR1L:CCP1CON<5:4>*0.1*16 = 62.5

i.e. CCPR1L:CCP1CON<5:4> = 62.5/1.6 = 39.0625= 39
